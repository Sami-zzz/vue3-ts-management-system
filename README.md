# vue3-ts-management-system

基于 Vite 开发的 Vue3 + TypeScript 后台管理系统项目

- Vue 3 Composition API
- Written in TypeScript

## 技术栈

- Axios
- Pinia
- Vue Router
- Element Plus
- ECharts
- ...

## 开始项目

```sh
#Project Setup
npm install

#Compile and Hot-Reload for Development
npm run dev

#Type-Check, Compile and Minify for Production
npm run build

```
